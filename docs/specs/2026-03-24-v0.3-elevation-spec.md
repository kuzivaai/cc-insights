# cc-insights v0.3.0 Elevation Spec

**Date**: 2026-03-24

## What Changed

### Bugs Fixed (v0.1.1)
- Removed unused `SessionData.errors` field and `ErrorEvent` type
- Rewrote `missing-build-cmd` to detect actual command patterns, not bare words
- Populated `StaleReference.section` in rule path (was always empty)

### Auto-Fix Architecture (v0.2.0)
- Added `InsightFix` type with line-level edit operations
- Added `applyFixes()` function for programmatic fix application
- Added inline suppression via `<!-- cc-insights-disable rule-id -->` comments
- VS Code: "Fix All" command, fix-on-save setting, data-driven quick-fixes
- Demoted informational rules to `info` severity (bloat, thin, tool-error, false-starts)

### New Rules (v0.3.0)
- `filler-phrase`: detects and rewrites padding phrases
- `stale-import`: detects broken @file imports
- `no-structure`: detects files with no markdown headings
- `redundant-config`: detects lines that restate package.json scripts

### CLI
- `npx @ccinsights/core CLAUDE.md` — lint
- `npx @ccinsights/core CLAUDE.md --fix` — lint and fix
- `npx @ccinsights/core CLAUDE.md --fix --dry-run` — preview fixes
